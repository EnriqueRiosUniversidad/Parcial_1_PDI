## Dataset

Este proyecto utiliza el siguiente dataset:

- **IQ-OTH/NCCD Lung Cancer Dataset (CT scans)**
- Fuente: Kaggle  
- Autor: Aditya Mahimkar  
- Link: https://www.kaggle.com/datasets/adityamahimkar/iqothnccd-lung-cancer-dataset  

Este dataset contiene imágenes de tomografía computarizada (CT) de cáncer de pulmón en formato JPG, utilizadas para evaluar algoritmos de mejora de contraste como CLAHE y transformadas Top-Hat.
